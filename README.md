## 项目描述

个人博客项目，点击访问[我的博客](https://www.sorvey.com)

### 技术栈：NestJS + postgresql + typeorm

## 已实现功能

- [x] restful Api，前后端分离，前端使用react
- [x] jwt权限策略
- [x] 用户管理
- [x] 文章分类及文章管理
- [x] 文章标签管理
- [x] 作品管理
- [x] 文章检索
- [x] 个人信息管理

## 待实现功能

- [x] 菜单及资源权限管理
- [x] 评论功能
- [x] ...

## 安装方法

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
